README - Guida all'Esecuzione della Pagina Web

Questo file README fornisce le indicazioni necessarie per eseguire correttamente il progetto della pagina web sul proprio computer.

Requisiti di Sistema

Per visualizzare e interagire con il progetto, sono necessari i seguenti componenti:

Un browser web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge o Safari).

Una connessione a Internet per scaricare il progetto dai repository indicati (le immagini e le risorse locali sono già presenti).

Struttura dei File

Il progetto è organizzato nella seguente struttura:

/ ─ index.html
  └ /css
      └ index.css
      └ articoli.css
      └ timeline.css
  └ /bilanci
      └ frasi.pdf
      └ 2023.pdf
      └ 2022.pdf
      └ …


  └ /immagini
      └ ambiente.jpg
      └ logo.svg
      └ modello.jpg
      └ …

index.html: File principale della pagina web.
REAADME.txt (questo file)

/css/index.css: File di stile per la personalizzazione dell'aspetto grafico generale degli elementi della pagina.
/css/articoli.css: File di stile per la personalizzazione dell'aspetto grafico dei testi presenti sotto le immagini principali della pagina.
/css/timeline.css: File di stile per la personalizzazione dell'aspetto grafico della sezione report di sostenibilità.

/bilanci: Cartella contenente i bilanci di sostenibilità in formato PDF ed un file js con i riassunti anno per anno.

/immagini: Cartella contenente immagini utilizzate nella pagina e il logo aziendale

Istruzioni per l'Esecuzione

Scaricare o clonare l'intero progetto in una cartella sul proprio computer.

Aprire il file index.html con un browser web supportato.

Navigare tra le diverse sezioni della pagina utilizzando la barra di navigazione.

Note Aggiuntive

Le immagini e i file PDF devono essere collocati correttamente nelle rispettive cartelle per garantire il corretto funzionamento dei collegamenti.

Per una migliore esperienza utente, assicurarsi di disporre di una risoluzione dello schermo adeguata e di un browser aggiornato.

Risoluzione dei Problemi

Se le immagini o i PDF non vengono visualizzati, verificare che i nomi dei file corrispondano esattamente a quelli specificati nelle directory.

Se ci fosero problemi di layout, potrebbe dipendere dalla compatibilità con il browser utilizzato.
